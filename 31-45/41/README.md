# Exercise No. 41

Define a function called `is_iterable()` that checks whether an object is iterable.


**Examples:**


    [IN]: is_iterable([4, 5])
    [OUT]: True


    [IN]: is_iterable('python')
    [OUT]: True


    [IN]: is_iterable(3.14)
    [OUT]: False


You don't need to implement validation in your solution.


**Note!** You only need to define the function.
